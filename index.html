How to Use

Gesture-controlled 3D workspace — use your hand in front of the camera.

✋ Gestures
☝️
Index Finger Up — Create Shape
Point your index finger straight up (other fingers curled). Hold ~0.3s to spawn the selected shape.
🤌
Pinch — Grab & Act
Bring thumb and index finger together over an object to grab and interact based on the active tool.
🛠️ Interaction Tools
✥
Move
Pinch an object and drag to move it through 3D space.
🔄
Rotate
Pinch an object and move your hand left/right or up/down to rotate it.
⤢
Scale
Pinch an object and move your hand up to grow it, or down to shrink it.
🗑️
Delete
Switch to Delete mode, then pinch an object to remove it with a shrink animation.
🔗
Combine
Switch to Combine mode, then pinch and drag an object into another to fuse them.
🧱 Shape Builder
🔷
Select a shape
Click Cube, Sphere, Cone, Cylinder, or Torus to set the active shape type.
☝️
Point up to spawn
After selecting a shape, raise your index finger to create it at the scene center.
💾 Scene Memory
💾
Save Scene
Stores all objects (position, rotation, scale, shape type) in browser local storage.
📂
Load Scene
Restores the last saved scene, clearing any current objects.
